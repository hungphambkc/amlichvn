jQuery.AmLich (Âm lịch Việt Nam)
======
**jQuery.AmLich** là một jQuery plugin được viết lại dựa trên mã nguồn của Hồ Ngọc Đức. Cách sử dụng:

```javascript
$(element).amLich({
  type: 'calendar', // month hoặc year hoặc text
  tableWidth: '200px' // Chiều rộng lịch
});
```
## Version
 * 0.1 - 24/07/2025 (dd/mm/yyyy)
 
## Đặc điểm
 * Âm và dương lịch
 * 24 tiết khí
 * Giờ đầu ngày
 * Giờ hoàng đạo
 * Các ngày lễ âm và dương lịch của Việt Nam
